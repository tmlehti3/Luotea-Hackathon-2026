# Alarms Dataset — Data Specification

**Dataset:** `alarms.csv`
**Last updated:** 2026-06-03

---

## Overview

This dataset contains **18,702 alarm events** from building management and fire safety systems at Valmet industrial sites. The data covers **January 2025 – May 2026** (~17 months) and includes HVAC alarms, fire alarms, fire equipment faults, line faults, and informational events.

Each row represents a single alarm event received by Luotea's alarm monitoring center. Events are generated by hardware devices (fire panels, building automation controllers, email-based alerting) and logged with timestamps, priority, alarm type, and a free-text description.

**Language:** Structured columns are in Finnish and English. The `EVENT_DESCRIPTION` field is in **Finnish**.

**No anonymization was required** — this dataset does not contain personal names, phone numbers, or email addresses.

## Notes for Participants

1. **One alert, many rows.** A single `ALERT_ID` can produce multiple rows — one per `LOCATION_ID`. Group by `ALERT_ID` for unique alarm events.
2. **LOOP values have leading spaces** (e.g., `"     1"`, `"  9999"`). Strip whitespace before analysis.
3. **96% of alarms have no work order.** The `WORKORDER_NO` link is sparse — most alarms are resolved without generating a formal work order.
4. **Priority is numeric but not sequential.** Values are 1, 13, 50, 60, 77, 99 — not a simple 1–N scale.
5. **EVENT_DESCRIPTION is in Finnish.** Contains technical alarm descriptions including device codes, temperatures, and system identifiers.
6. **Encoding:** UTF-8 with BOM.
